SrvrPowerCtrl Log


These log entries can assist in troubleshooting problems with SrvrPowerCtrl. Generally speaking, it's best to set SrvrPowerCtrl's logging to 'Debug' in order to diagnose problems. You can set SrvrPowerCtrl's logging level at the bottom of the settings page. SrvrPowerCtrl's current logging level is set to: '[% srvrpowerctrl_logLevel %]'.
